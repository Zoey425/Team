시맨틱 태그 사용 이유

  • 검색엔진 최적화(SEO) : 검색엔진이 알맞은 검색결과를 내기 위해 웹사이트를 크롤링할 때 웹사이트의 내부에 담긴 정보를 토대로 사이트를 분석
  • 쉬운 소스코드 구조화 : 소스코드가 구조화 되어있어 분석에 용이하다
  • 코드 가독성 향상 : 유지보수, 코드 공유 등 굳이 클래스 지정하지 않아도 어느 영역인지 쉽게 알수 있음

웅진씽크빅, 웅진 IT 홈페이지 HTML 분석 및 시맨틱 태그 사용해야하는 이유 공유(개인의견)

  • 박은영 : 웅진씽크빅은 div 태그로만 구성되어있다보니 가장 최상단에 묶여있는 코드를 찾으려고 할 때 시간이 소요된다. 반면 웅진IT에 시맨틱 태그로 사용되어있어서 section 별로 묶여 있어 영역이 명확하게 들어나서 영역별 코드를 찾아보기 쉬웠다.
  • 신수현 : 웅진IT는 구역을 시맨틱 태그로 나누어져 있었고, 웅진씽크빅은 div 태그로만 구성되어있었다. 세부적으로 웅진씽크빅의 gnb는 ul과 li 태그로 구성되어있지만, 웅진IT는 div 태그로만 구성되어있었다. 구역뿐만 아니라 세부적인 요소도 시맨틱 태그를 적용하면 더 가독성이 좋을 것 같다.
